# 🎓 Student Record Management System

A clean, modern, and modular client-side web application designed to track, manage, and analyze student academic records efficiently. 

---

## 🚀 Features
* **Full CRUD Operations:** Seamlessly add, view, update, and delete student records directly from a dynamic user interface.
* **Persistent Storage:** Uses browser `LocalStorage` to save records dynamically, ensuring data is never lost on page refresh.
* **Smart Searching & Filtering:** Real-time lookup utility lets you filter students instantly by Name or unique ID.
* **Academic Sorting Analytics:** Advanced sorting logic organizes student lists alphabetically (A-Z) or dynamically tracks top performers by highest Grade/GPA.
* **Built-in Exception Logic:** Front-end validations prevent empty entries and safeguard against duplicate Unique Student ID conflicts.

---

## 🛠️ Tech Stack
* **Structure:** HTML5
* **Styling & Theme:** CSS3 (Modern Flexbox/Grid layout)
* **Core Logic & Engine:** Vanilla JavaScript (ES6+)
